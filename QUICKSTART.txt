================================================================================
FIRST TIME USE
================================================================================

1. Select directories to search:
   - Check boxes in left panel (Documents, Downloads, etc.)
   - Tri-state: Check parent to select all children

2. Enter search term:
   - Type filename or pattern in top search bar
   - Toggle "Case Sensitive" if needed

3. Click "Search" button

4. Browse results:
   - Results organized in tabs by file type
   - Click column headers to sort
   - Select multiple files (Ctrl+Click or Shift+Click)

5. Perform actions:
   - "Open" - Opens selected files
   - "Open Location" - Opens folder in Explorer
   - "Copy To..." - Copies files to destination
   - "Move To..." - Moves files to destination
   - "Clear Results" - Removes all results

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "ModuleNotFoundError: No module named 'PyQt6'"
Solution: Run: pip install PyQt6

Problem: Search is slow
Solution: Select fewer directories or use specific search term

Problem: Permission errors
Solution: Avoid system directories (C:\Windows, Program Files)
          or run as Administrator

Problem: Files won't open
Solution: Check Windows file associations

================================================================================
TIPS & TRICKS
================================================================================

1. Use specific search terms for faster results

3. Use dark mode for extended use (less eye strain)
