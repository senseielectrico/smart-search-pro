================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: "ImportError: No module named 'PyQt6'"
SOLUTION: pip install PyQt6

ISSUE: "ModuleNotFoundError: No module named 'win32com'"
SOLUTION:
  pip install pywin32
  python -m pip install --upgrade pywin32
  python Scripts/pywin32_postinstall.py -install

ISSUE: Application crashes on startup
SOLUTION:
  1. Run: start.bat --console
  2. Check error output
  3. Check log: %USERPROFILE%\.smart_search\error.log
  4. Run: python verify_setup.py

ISSUE: Windows Search not available
SOLUTION:
  - Normal on some systems
  - Application falls back to filesystem search
  - May be slower, still functional

ISSUE: Slow performance
SOLUTION:
  - Reduce search scope (fewer directories)
  - Disable content search (use filename only)
  - Index directories in Windows Search settings
  - Check system resources

================================================================================
PERFORMANCE OPTIMIZATION
================================================================================

FOR FASTER SEARCHES:

1. Index Directories
   - Windows Settings > Search
   - Add commonly used folders to index
   - Faster Windows Search API queries

2. Reduce Search Scope
   - Select only necessary directories
   - Avoid searching entire drive
   - Focus on target folders

3. Use Filename Only
   - Content search is slower
   - Use only when necessary
   - Filename search much faster

4. System Maintenance
   - Close other applications
   - Check available RAM
   - Check disk space
   - Run disk cleanup

BENCHMARKS:
  - Filename search: 1000s of files in seconds
  - Content search: Slower, depends on file sizes
  - Windows indexed folders: Fastest
  - Non-indexed folders: Slower
