QUE HACE ESTA APLICACION
================================================================================

Smart Search es un buscador avanzado de archivos para Windows que:

✓ Busca archivos por nombre y contenido
✓ Usa la API de Windows Search (rapido)
✓ Organiza resultados por tipo de archivo
✓ Permite copiar/mover archivos
✓ Tiene tema claro y oscuro

COMO EMPEZAR A USAR
================================================================================

1. Doble clic en: run_smart_search.bat
