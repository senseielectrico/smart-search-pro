========================================
INSTALLATION
========================================

1. Install ExifTool:
   - Download from: https://exiftool.org/
   - Extract to: D:\MAIN_DATA_CENTER\Dev_Tools\ExifTool\
   - Or add to system PATH

2. Verify installation:
   python test_exiftool_integration.py

3. Optional: Install gpxpy for GPS geotagging:
   pip install gpxpy
